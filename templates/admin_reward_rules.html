How Reward Rules Work

Contribution Multiplier: Users earn this many points per R1 contributed. Set to 1 for 1:1 ratio.

Welcome Bonus: Automatically given to new users upon registration.

Referral Bonus: Given when a user successfully refers another user to the platform.

Monthly Bonus: Given monthly to users who made contributions in the previous month.

Event Attendance: Given for attending stokvel events and meetings.

Loan Payment: Given for on-time loan payments and good financial behavior.
